Persoonlijk Samenvatting

Ik ga de Front-End kant op, maar ik ben ook bezig met het leren van Back-End. Ik kan goed samenwerken met andere mensen en goed omgaan met wie dan ook. Mijn hobby's zijn; sporten, gamen, coderen en reizen. Ik wil graag in een team werken waar ik veel kan leren van de andere mensen in de groep, terwijl ik ook de bedrijf en de team help met hun doelen. Ik heb tot mijn 17'de op hoog niveau gevoetbald. Ik heb zowel op school als daar heel veel geleerd over hoe je moet samenwerken als een team. Als je naar beneden scrollt zie je sommige projecten die ik heb gemaakt, buiten die projecten heb ik nog meer projecten, alle projecten die ik heb gemaakt kan jij zien op mijn github pagina.
